# array-grouper
Small function to split an array up into groups of n size.

# How to run

The easiest way to run this code is to execute it via node:
`node index.js` which will give the following output:

```javascript
A: [ [ 1, 2, 3 ], [ 4, 5, 6 ] ]
B: [ [ 1, 2, 3 ], [ 4, 5, 6 ], [ 7 ] ]   
C: [ [ 1, 2, 3 ], [ 4, 5, 6 ], [ 7, 8 ] ]
D: [ 1, 2, 3, 4, 5, 6 ]
E: []
